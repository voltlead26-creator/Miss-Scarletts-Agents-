## Overview
This prompt builds a comprehensive performance measurement framework for AI marketing agents and tools — quantifying productivity gains, pipeline contribution, cost savings, and quality outcomes so marketing leaders can prove AI ROI, allocate AI budgets intelligently, and optimize underperforming agents. Use it quarterly to run a full AI stack audit or weekly to track active agent performance against targets.

## Module 4: Agent Performance Scorecard

## Integration Tips

- **HubSpot:** Create a custom property "AI Tool Source" on Contacts and Deals. Tag every AI-influenced activity at point of creation. Build a report: Deals by AI Tool Source × Stage for live pipeline attribution.
- **Salesforce:** Add "AI Touchpoint" checkbox to Campaign Member and Activity objects. Build a SOQL report grouping closed-won by AI touchpoint count to isolate AI-influenced revenue.
- **Google Sheets:** Use the ROI model template — copy the TCAO table into a Sheets tracker with monthly columns. Set up conditional formatting to flag tools where cost exceeds measured value for the past 60 days.
- **Notion:** Build a living AI Stack Wiki — one page per tool with performance data updated monthly. Link each tool page to its scorecard output. Use Notion AI to synthesize quarterly trends automatically.
- **Zapier / Make:** Automate data collection: schedule a monthly Zap that pulls Salesforce pipeline data, HubSpot activity counts, and tool cost data from billing APIs into a Google Sheet. Feed that Sheet to this prompt for automated monthly AI ROI reports.

## Troubleshooting

**Problem: Can't isolate AI-influenced pipeline because we don't tag AI activities in CRM**
Solution: Implement UTM taxonomy and CRM tagging retroactively for the past 90 days using campaign naming conventions. For future measurement, add "AI-assisted: yes/no" as a mandatory field on all Campaign Member records and Sequences. Start with this quarter as your clean baseline — even one quarter of clean data is enough for a statistically meaningful ROI report.

**Problem: The AI tool vendor claims much higher ROI than our internal measurement shows**
Solution: Vendor ROI numbers use correlation, not causation (they count all pipeline in accounts that use their tool, whether or not the tool influenced the deal). Apply a 0.3–0.5 attribution confidence factor to any vendor-supplied pipeline numbers. Build your own control group: run identical campaigns with and without the AI tool for 30 days and compare conversion rates. Your measured delta is the true causal impact.

**Problem: Team resistance — marketers feel this audit threatens their jobs**
Solution: Frame the report as "where should AI do more so humans can do higher-value work" rather than "what can AI replace." Involve team leads in the measurement methodology design. Share findings transparently. When an AI tool underperforms, celebrate the human work it failed to match — this builds psychological safety for honest reporting.
