## Integration Tips

- **Salesforce / HubSpot CRM:** Attach the evaluation framework as a deal asset in your CRM opportunity record. Use deal tags to track which evaluation version is deployed per account. Trigger automated Slack alerts to PMM when a framework is opened by 3+ stakeholders (indicating active committee evaluation).
- **Notion / Confluence:** Host the buyer-facing version in a shared workspace with commenting enabled — buying committees can score vendors collaboratively and you get visibility into their deliberations via notification alerts.
- **Mutually Agreed-Upon Success Plans (MASPs):** Embed the POC scorecard directly into your shared success plan tool (Salesforce, DealHub, or Notion deal rooms) so POC criteria are co-owned by buyer and seller.
- **Google Sheets / Airtable:** Convert the weighted matrix into a shared spreadsheet with auto-calculated scoring. Buyers enter scores; formulas compute totals. Create a shareable link and track access via Google Analytics or Bit.ly.
- **Seismic / Highspot / Showpad:** Publish the framework as a sales enablement asset with version control. Tag by industry, deal size, and competitor set so AEs surface the right version automatically.
- **Zapier / Make.com:** Build a workflow: when AE marks deal as "Competitive Evaluation" stage in CRM → automatically send buyer the framework via email sequence → notify PMM Slack channel → log asset delivery in CRM activity.

## Troubleshooting

**Problem:** Buying committee refuses to use a vendor-provided framework, citing bias concerns.
**Solution:** Reframe delivery. Instead of "here's how to evaluate vendors," send it as a third-party-style "B2B [Category] Selection Guide" with neutral language. Remove your logo from the top-level document and place it only in a footer. Consider co-branding with a trusted analyst or publishing through a neutral channel (LinkedIn article, industry association) so it enters the conversation organically. Your champion can then "discover" it and share internally.

**Problem:** Competitors are gaming the scorecard criteria by coaching prospects on how to answer evaluation questions favorably.
**Solution:** Shift 30%+ of criteria to POC-verified metrics rather than self-reported claims. Objective test results (sync latency measured in seconds, accuracy tracked over a 2-week POC, implementation timeline with actual go-live date) cannot be faked in a controlled environment. Add "requires third-party verification" flags to all high-weight criteria.

**Problem:** Framework is too complex for SMB deals where buyers want a simple 3-vendor comparison, not a 100-point matrix.
**Solution:** Maintain two versions: the full enterprise framework (100-point matrix for deals >$50K ACV with formal evaluation processes) and a "quick comparison card" (10-point checklist, one page, designed for 30-minute decisions). Use deal size and committee size as triggers for which version the AE deploys. The quick card should still embed your top 3 knockout criteria.
