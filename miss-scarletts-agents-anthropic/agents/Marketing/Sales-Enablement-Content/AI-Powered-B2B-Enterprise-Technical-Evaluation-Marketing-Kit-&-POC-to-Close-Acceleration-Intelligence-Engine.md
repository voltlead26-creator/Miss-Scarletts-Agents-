## Troubleshooting

**Problem: Rep says "the security package doesn't match our actual certifications"**
Solution: Module 2 is a template framework — the first deployment requires your InfoSec team to validate specific certifications, data practices, and escalation contacts. Schedule a 30-minute InfoSec + PMM review session to customize the package once; then it's reusable across all enterprise deals. Consider building a living Security Trust Center document that auto-exports from your security attestation tools (Vanta, Drata, Secureframe).

**Problem: Champion won't use the ROI business case because the numbers feel "too salesy"**
Solution: Reframe the ROI model as a "mutual business case" built together in a joint working session. Replace bold revenue projections with champion-inputted assumptions — when they enter their own data, they own the numbers. Run a 45-minute "business case workshop" call in Week 3 of evaluation where you build the model live with the champion. Their inputs = their narrative = their internal sell.

**Problem: Evaluation stalls at the security review stage and the kit isn't accelerating it**
Solution: The security package accelerates self-serve review, not backlogged IT teams. The real unlock is a dedicated "Security Fast-Track Call" — a 60-minute live session between your CISO/Security Engineer and their IT security team to answer questions in real time. Offer this proactively in Week 2. Simultaneously, offer to pre-fill their specific security questionnaire (SIG Lite, CAIQ, or custom) — this alone reduces security review time by 50-70% in enterprise deals.
