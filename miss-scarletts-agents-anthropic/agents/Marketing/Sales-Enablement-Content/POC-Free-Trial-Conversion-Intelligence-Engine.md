## Integration Tips

- **HubSpot:** Use Workflow triggers on contact property "Trial Milestone Score" to auto-enroll in the correct activation track. Build custom properties for each MEDDPICC dimension and create a deal score property that rolls up automatically.
- **Outreach / Salesloft:** Map each activation track to a dedicated Sequence. Use "A/B Sequence" testing on Day 3 and Day 7 subject lines. Set auto-pause rules when a prospect books a meeting or replies.
- **Pendo / Mixpanel / Amplitude:** Create a "Trial Conversion Funnel" report tracking each milestone event. Build a Slack alert that fires to the AE when a prospect achieves the "revenue attributed" milestone or when they go >3 days without login.
- **Salesforce:** Add a "Trial Health Score" field (0-100) calculated from product telemetry. Build a Flow that creates a Task for the AE when score drops below 40. Map MEDDPICC fields directly to opportunity record for pipeline review hygiene.
- **Gong:** Tag "trial review" calls for AI analysis. Build a scorecard that checks whether AE coached champion presentation, confirmed ROI numbers, and used transition language to commercial discussion.
- **Zapier / Make:** Build an automation: when "Evidence Package sent" = true AND "Trial end date" within 3 days → create DocuSign envelope with pre-populated MSA and send to Economic Buyer contact.

## Troubleshooting

**Problem: Prospects start the trial enthusiastically but go dark after Day 3.**
Solution: Almost always an activation failure — they haven't reached their first "aha moment." Audit your Day 1-3 sequence for friction. Add a SE-staffed "Quick Start" call offer in the Day 2 touch. The offer alone (even if not taken) signals white-glove support and reduces anxiety. Check whether your integration setup has a hidden complexity creating a 24-72 hour delay in first value.

**Problem: Champion is engaged but can't get economic buyer approval during the trial.**
Solution: The champion kit is incomplete or the ROI case isn't quantified precisely enough for finance. Send the champion the pre-populated executive summary and offer to join a 20-minute "investment overview" call with them and the economic buyer together — position it as a "trial outcomes briefing" not a sales call. Provide a "cost of inaction" calculation (what it costs them per month to delay), which makes the urgency tangible for finance.

**Problem: Trial-to-close rate is high but sales cycle after trial is too long.**
Solution: The commercial discussion is starting too late. Build a soft commercial signal into Week 2 — the AE mentions "assuming this goes well, our typical customers start onboarding in [timeframe]" to plant the expectation. Introduce the paper process question by Day 14: "Just so we're not surprised — have you run software purchases through procurement before? What does that process typically look like?" Parallel-tracking legal/procurement saves 2-4 weeks of post-trial cycle time.
