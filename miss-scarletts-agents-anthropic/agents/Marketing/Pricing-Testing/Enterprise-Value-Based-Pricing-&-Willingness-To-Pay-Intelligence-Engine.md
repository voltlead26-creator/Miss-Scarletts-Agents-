## Success Metrics

- Value multiple is 5–10x: For every $1 you charge, the customer should receive $5–10 in quantifiable return. Below 5x indicates underpricing; above 10x indicates significant pricing power being left on the table.
- Discount rate drops below 15%: When sales can articulate value in CFO language, discounting becomes unnecessary. Track average discount rate before and after implementing this framework.
- ACV increases 20–40% in the first two renewal cycles using value-based packaging redesign.
- Win rate on price-objection deals improves: Track deals where "price is too high" was raised vs. closed-won rate. Target: increase from baseline by 25%.
- Expansion revenue grows as % of ARR: Value-metric pricing naturally drives NRR above 110%. Track quarterly.

## Integration Tips

- **Salesforce / HubSpot CRM**: Embed the ROI Statement Template as a custom field in the opportunity record. When an AE fills in discovery inputs, auto-generate a deal-specific business case using a Salesforce Flow or HubSpot Workflow + Claude API. Attach the PDF to the opportunity before first pricing call.
- **Google Sheets / Notion**: Build the WTP segmentation matrix in a shared doc and connect it to your CRM via Zapier so reps see the relevant WTP range and value defense script based on company size detected from enrichment tools (Clearbit, Apollo, Clay).
- **CPQ (Configure Price Quote) Tools like DealHub, Conga, or Salesforce CPQ**: Use the packaging architecture and discount authority matrix to configure hard stops and approval workflows directly in CPQ. This prevents reps from discounting beyond thresholds without a documented value exchange.
- **Gong / Chorus**: After deploying value defense scripts, use Gong's tracker feature to monitor how often reps are using value language vs. resorting to discount concessions. Review weekly in sales leadership team to reinforce behavior.
- **Stripe / Chargebee / Recurly**: If switching to a value metric (e.g., pipeline-based pricing), configure a metered billing integration that reads from your product's data warehouse and generates invoices automatically. Connect to Slack to alert CSMs when a customer is approaching tier limits—creating a natural expansion conversation trigger.

## Troubleshooting

- **Problem:** The ROI calculation feels too theoretical—prospects don't trust the numbers.
  **Solution:** Replace internal projections with third-party validation. Use Forrester TEI study language, G2 review quotes with specific metrics, or reference customer case studies with verified ROI. Frame it as "conservative estimate based on results from [3 customers similar to you]" rather than a promise.

- **Problem:** Sales team isn't using the value defense scripts in calls and reverts to discounting.
  **Solution:** The scripts are too long to memorize. Run the Advanced Version prompt again with the instruction "compress each negotiation defense into a 2-sentence response the rep can say naturally." Then build Gong Call Cards or HubSpot Deal Intelligence snippets with these compressed scripts. Practice in weekly role-play before discount requests hit.

- **Problem:** Enterprise procurement comes back with a legal/procurement price benchmark that's 30% below your price.
  **Solution:** The ROI model lacks procurement-level proof. Run Deliverable 3 again and specifically add a "Total Cost of Ownership comparison vs. build-in-house or competitive alternative" section. Procurement departments respond to TCO, not feature comparisons. Include implementation costs, time-to-value, and integration costs in the comparison—your TCO advantage often narrows the perceived price gap significantly.
