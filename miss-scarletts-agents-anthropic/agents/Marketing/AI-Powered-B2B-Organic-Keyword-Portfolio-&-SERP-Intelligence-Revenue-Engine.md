# ANALYTICAL FRAMEWORKS TO APPLY

**1. KEYWORD REVENUE TIERING (KRT) MODEL**
Classify every keyword in the portfolio across two dimensions:
- Commercial Intent Score (1–5): 1=Pure informational, 3=Research/comparison, 5=High purchase intent
- Position Value Score (1–5): 1=Position 21+, 3=Position 6–15, 5=Position 1–3

Plot each keyword in a 2x2 matrix:
- Quadrant 1 (High Intent + Strong Position) = PROTECT & SCALE — these are your revenue engines
- Quadrant 2 (High Intent + Weak Position) = ATTACK — highest ROI investment targets
- Quadrant 3 (Low Intent + Strong Position) = NURTURE — pipeline influencers, optimize for conversion path
- Quadrant 4 (Low Intent + Weak Position) = HARVEST or ABANDON — evaluate if brand-building justifies investment

**2. SERP FEATURE CAPTURE ANALYSIS**
For each keyword cluster, map which SERP features are present and owned:
- Featured Snippet: Is our content formatted to capture it? (Use inverted pyramid structure, direct answer in first 40–60 words, table/list format where applicable)
- AI Overview (Google): Is our content cited? Check E-E-A-T signals, structured data, and direct answer formats
- People Also Ask: Are we capturing PAA boxes? Map PAA questions to existing or missing content
- Knowledge Panel / Brand Entity: Is brand entity markup complete?
- Video Carousel: Where could video content capture additional SERP real estate?
- Sitelinks: Are structured internal links supporting sitelink eligibility for branded terms?

**3. COMPETITIVE DISPLACEMENT FORENSICS**
For every keyword where a competitor gained 3+ positions in the last 30–90 days:
- Identify the trigger: new content published, existing page updated, link building surge, or technical improvement
- Estimate traffic and pipeline impact of the displacement
- Score effort-to-reclaim: Low (on-page refresh), Medium (content expansion), High (new content + links needed)
- Prioritize reclaim actions by revenue impact

**4. KEYWORD CANNIBALIZATION & INTENT MISMATCH AUDIT**
- Cannibalization: Identify where 2+ pages target the same keyword — consolidate or differentiate
- Intent mismatch: Flag where the ranking page is informational but the search intent is transactional (or vice versa) — mismatches suppress CVR regardless of ranking
- Stage mismatch: Flag where a decision-stage keyword ranks an awareness-stage page — recommend conversion path optimization

**5. LONG-TAIL REVENUE AMPLIFICATION MODEL**
Segment long-tail keywords (500 monthly searches or under) by:
- Buyer specificity: Does this keyword suggest a specific role, pain, or use case? High specificity = higher CVR
- Cluster membership: Does this keyword belong to a high-performing cluster we can amplify?
- Content gap: Is there an existing page that could absorb this keyword with minimal effort?

Apply the Long-Tail Revenue Multiplier: even low-volume high-intent keywords compound significantly across a portfolio. Model aggregate impact of capturing 100 long-tail keywords at 3% CVR vs. 10 head terms at 0.5% CVR.

**6. AI SEARCH PRESENCE AUDIT**
For top 20 commercial-intent keywords:
- Test brand citation in ChatGPT, Perplexity, Claude, and Google AI Overviews (document results)
- Score: Cited prominently / Cited peripherally / Not cited
- For non-cited keywords: identify what competitors are cited and what content structure differences explain it
- GEO Optimization Recommendations: Authoritative definition pages, structured comparison content, cited statistics, and FAQ schema

## Integration Tips

- **Google Search Console:** Export "Performance > Search Results" report with queries, clicks, impressions, CTR, and position. Filter by country if needed. This is your primary input data source.
- **Ahrefs / Semrush:** Pull organic keyword position history, competitor SERP overlap reports, and keyword gap analysis. Export to CSV and paste into prompt for maximum analysis depth.
- **GA4:** Pull organic landing page report (Sessions, Conversions, Revenue by Landing Page). Join with Search Console data in Looker Studio for full keyword→conversion view before pasting.
- **HubSpot / Salesforce:** Use UTM tracking and "Original Source = Organic Search" filter to pull MQL and pipeline data by landing page. Map to keyword clusters for revenue attribution.
- **Looker Studio Automation:** Build a Looker Studio report that auto-pulls GSC + GA4 + CRM data weekly. Run this prompt monthly against the latest export for continuous portfolio intelligence.
- **Zapier / Make:** Set up automated alerts when any top-20 commercial keyword drops 3+ positions. Trigger this prompt's "Competitive Displacement" analysis automatically when displacement is detected.
- **Screaming Frog + AI:** Crawl site to identify cannibalizing pages, then feed URL list + keyword targets into the Cannibalization section of this prompt for automated consolidation recommendations.

## Troubleshooting

**Problem:** Search Console data doesn't show keyword-level conversion data — you can only see clicks and impressions.
**Solution:** Use GA4 to identify your top organic landing pages by conversion rate. Cross-reference those pages with their primary keywords in Search Console using the "Pages" tab to filter queries per page. This gives you an approximate keyword→conversion mapping. For more precision, use UTM parameters on internal links from organic landing pages to conversion pages to track full journey in GA4.

**Problem:** You have 2,000+ keywords and the AI analysis becomes generic rather than specific.
**Solution:** Segment your portfolio first. Run three separate prompts: (1) Commercial Intent keywords only (positions 1–20), (2) Comparison/Research keywords (positions 1–30), (3) Informational keywords (positions 1–10 that drive significant traffic). The commercial intent analysis will be the highest ROI starting point.

**Problem:** Competitor position data seems unreliable — rank trackers show different numbers than what you see in actual search results.
**Solution:** Rank trackers use data center sampling and personalized results are excluded — expect 1–3 position variance as normal. For high-stakes keywords, manually check rankings in incognito/private mode from the target geography. For the AI analysis, note the tracking tool used and flag that position data has ±2 position margin of error. Focus analysis on relative trends (gaining vs. losing) rather than absolute positions.
