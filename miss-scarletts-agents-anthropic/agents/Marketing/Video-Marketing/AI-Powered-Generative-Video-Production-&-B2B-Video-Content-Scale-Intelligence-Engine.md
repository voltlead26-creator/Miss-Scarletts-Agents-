## Troubleshooting

**Problem: AI-generated avatar looks "uncanny valley" and B2B buyers don't trust it.**
Fix: Use HeyGen's Photo Avatar feature with your actual executive's photo + voice clone (ElevenLabs) — gives authentic-looking video without full recording sessions. Alternatively, use AI for b-roll only and record real human spokesperson for the main camera; AI does the production work, not the face. Always disclose AI-assisted production in content description to build trust.

**Problem: AI b-roll doesn't match brand aesthetic — looks generic or wrong industry.**
Fix: Build a detailed negative prompt library and run 10 test generations before any campaign. Use Runway's Reference Image feature to train consistent visual style. Create a "b-roll approval library" of 50+ pre-approved AI clips that match brand standards — production team pulls from library rather than generating fresh for every video.

**Problem: Video production velocity is high but pipeline impact isn't showing up in attribution.**
Fix: This is a tracking gap, not a content gap. Audit: (1) Is Vidyard/Wistia properly integrated with HubSpot/SFDC? (2) Are UTM parameters firing on all video landing pages? (3) Is the video attribution window long enough? (B2B sales cycles are 3-6 months — set attribution window to 180 days.) (4) Are SDRs logging video sends in Salesforce? Start with tracking fix before creating more content.
