# AI-Powered B2B SaaS Organic Search Performance Analytics & Content-to-Pipeline Revenue Attribution Intelligence Engine

**Difficulty:** Advanced | **Time:** 30 min | **Tags:** b2b, seo, analytics, revenue-attribution, pipeline, content-performance

## Overview
Transforms raw organic search data from Google Search Console, GA4, and your CRM into a board-ready revenue contribution report — mapping every keyword, landing page, and content asset to pipeline created, pipeline influenced, and closed-won revenue. Use this prompt monthly or quarterly to prove SEO's business impact, prioritize content investment, and identify the highest-ROI organic search opportunities.

## Quick Copy-Paste Version

You are a senior B2B SaaS marketing analyst specializing in organic search revenue attribution. Analyze the following organic search data and produce a complete performance report with revenue attribution.

DATA TO ANALYZE:
- Google Search Console data: [paste top 50 queries by clicks, impressions, CTR, position for last 90 days]
- GA4 organic sessions by landing page: [paste top 30 pages by sessions, engagement rate, goal completions]
- CRM pipeline data: [paste leads/MQLs/pipeline opportunities with original source = organic search, last 90 days]
- Current keyword rankings: [paste rank tracking data for target keywords]
- Competitor domain: [competitor 1], [competitor 2]

DELIVER THIS ANALYSIS:

1. EXECUTIVE SUMMARY (3 bullets max)
   - Organic search contribution to total pipeline this period ($X pipeline, X% of total)
   - Top 3 wins and top 3 gaps vs. prior period
   - Single most important action to take this quarter

2. PIPELINE ATTRIBUTION WATERFALL
   Build this funnel with conversion rates at each stage:
   Organic Sessions → Engaged Sessions → Leads → MQLs → Opportunities → Pipeline $ → Closed-Won $
   Flag where the biggest drop-offs occur and what that means.

3. TOP 10 REVENUE-GENERATING CONTENT ASSETS
   For each: URL | Monthly Sessions | Lead Conv% | Pipeline Influenced $ | Rank for Primary Keyword | Action (optimize/promote/update/sunset)

4. KEYWORD PORTFOLIO HEALTH SCORECARD
   Segment keywords into 4 buckets:
   - Brand keywords (protect & grow)
   - Commercial intent non-brand (highest pipeline value — prioritize)
   - Informational/top-of-funnel (build awareness, nurture to pipeline)
   - Competitor/comparison keywords (displacement opportunity)
   Report performance and gaps in each bucket.

5. COMPETITIVE ORGANIC SHARE OF VOICE

   - Our current position vs. [competitor 1] and [competitor 2]
   - Estimated monthly organic traffic we're winning vs. losing to competitors

6. CONTENT-TO-REVENUE ATTRIBUTION BY TOPIC CLUSTER
   Map performance to your topic clusters:
   - Which clusters drive the most pipeline?
   - Which clusters have the highest keyword rankings but lowest pipeline conversion (content-pipeline gap)?
   - Which clusters are underinvested relative to commercial opportunity?

7. 30/60/90-DAY ORGANIC GROWTH ROADMAP
   Specific, sequenced recommendations with projected pipeline impact for each action.

Output format: Executive brief with tables. Every recommendation must include expected pipeline impact and effort estimate (S/M/L).

## Integration Tips

- **Google Search Console → GA4 linking:** Enable GSC-GA4 integration to see keyword data alongside on-site behavior. Export blended reports via GA4's "Search Console" exploration for the richest dataset to paste into this prompt.
- **HubSpot/Salesforce CRM:** Use UTM parameters (utm_source=organic, utm_medium=organic) to segment pipeline reports. Pull "Original Source = Organic Search" deals from your CRM for pipeline attribution data. Run this as a saved report refreshed monthly.
- **Semrush/Ahrefs:** Export the "Organic Positions" and "Organic Pages" reports for both your domain and each competitor. The Position Tracking tool provides the ranking velocity data needed for Phase 2.2.
- **Looker Studio / Tableau automation:** Build a Looker Studio dashboard that auto-pulls GSC + GA4 + CRM data and feeds a structured table that you paste directly into this prompt each month — reducing report prep time from 4 hours to 20 minutes.
- **Zapier/Make.com workflow:** Set up an automated monthly workflow that pulls GSC data via API, CRM pipeline data, and rank tracking exports, compiles them into a structured Google Sheet, and pings your team to run the prompt analysis within 48 hours of month-end.
- **Notion integration:** Store prompt outputs in a Notion database with tags for content type, keyword cluster, and action priority. Link to individual content assets in your content inventory for a complete content performance wiki.

## Troubleshooting

**Problem: Attribution data is unreliable — organic pipeline numbers don't match what the sales team sees.**
Solution: This is the most common issue. Start by auditing your UTM tagging strategy — check whether any organic landing pages are mis-tagged as direct or referral. Use Google Tag Manager to add a session storage variable that captures the first organic landing page for all sessions. In the prompt, use a note like "our current attribution likely undercounts organic by 30-40%" and apply a correction factor based on last-touch vs. first-touch discrepancy. Then prioritize fixing tagging before the next reporting cycle.

**Problem: The prompt produces generic recommendations that aren't specific to our business.**
Solution: The richer your input data, the more specific and actionable the output. Ensure you're providing: (1) real GSC data with actual queries and positions rather than summary metrics, (2) CRM pipeline data by content asset or at minimum by channel, and (3) specific competitor names and their top-ranking URLs for your category keywords. Also add your ICP description and ACV — this dramatically improves the prioritization logic.

**Problem: The competitive SOV analysis is outdated or inaccurate.**
Solution: Organic competitive data from Semrush/Ahrefs has a 2-4 week lag and traffic estimates have ±30% accuracy. State this limitation explicitly in the prompt input: "Note: competitor data is estimated — treat SOV numbers as directional, not precise." For higher-accuracy competitive analysis, manually check SERP positions for your top 20 keywords using a rank tracking tool on a weekly basis and use those verified positions in the prompt rather than third-party estimates.
