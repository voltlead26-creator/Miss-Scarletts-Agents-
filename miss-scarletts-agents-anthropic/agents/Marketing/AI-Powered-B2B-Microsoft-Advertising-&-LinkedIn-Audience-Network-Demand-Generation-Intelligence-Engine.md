## Integration Tips

- **Salesforce/HubSpot:** Use Microsoft Ads' native HubSpot and Salesforce integrations (available under Tools → Conversions → CRM Import) to import offline conversion events. Map closed-won deals back to Microsoft Ads click IDs for revenue attribution.
- **LinkedIn Insight Tag:** Deploy LinkedIn's Insight Tag alongside Microsoft's UET tag — this builds LinkedIn-matched audiences from website visitors which can then be targeted in Microsoft Ads' LinkedIn Profile Targeting UI.
- **Google Ads Sync:** Export your Google Ads negative keyword lists monthly and import into Microsoft Ads. Also mirror your Google Ads campaign structure in Microsoft Ads to enable apples-to-apples performance comparison.
- **Zapier/Make:** Create automated alerts when Microsoft Ads CPL exceeds target — trigger Slack notifications or Google Sheets logging using the Microsoft Advertising API webhook.
- **HubSpot Workflows:** Build a triggered workflow that tags contacts with `Microsoft Ads: Clicked` when UTM source = bing enters CRM, enabling separate nurture sequence enrollment and pipeline velocity tracking.
- **Power BI / Looker Studio:** Microsoft Ads has a native Looker Studio connector — build a cross-channel dashboard combining Google Ads, LinkedIn Ads, and Microsoft Ads pipeline data in one view.

## Troubleshooting

**Issue: LinkedIn Profile Targeting audiences are too small (under 300 people) and campaigns won't serve**
*Fix:* Broaden one targeting dimension at a time. Start by removing the company-size filter, then loosen seniority to include "Associate" level if needed. Alternatively, combine multiple audience segments into one campaign using OR logic rather than stacked AND filters. Microsoft requires a minimum of 300 matched users in any MSAN audience.

**Issue: Smart bidding (Target CPA) is spending erratically or not spending at all**
*Fix:* Smart bidding needs a minimum of 30-50 conversions in a 30-day window to optimize effectively. In the first 30 days, use Enhanced CPC or Maximize Conversions without a target CPA cap. Once you have 30+ conversions, set a Target CPA at 1.5x your actual CPL to give the algorithm room to learn before tightening.

**Issue: MSAN placements driving high click volume but zero conversions (content network fraud)**
*Fix:* Pull the placement report under Reports → Run Report → Website URL. Exclude any domain with CTR > 2% but 0 conversions, any gaming/entertainment domains, and any domain serving <10 impressions (long-tail fraud). Build a master exclusion list of 200+ irrelevant/low-quality domains in the first 2 weeks. Set a URL exclusion at the account level, not just campaign level.
