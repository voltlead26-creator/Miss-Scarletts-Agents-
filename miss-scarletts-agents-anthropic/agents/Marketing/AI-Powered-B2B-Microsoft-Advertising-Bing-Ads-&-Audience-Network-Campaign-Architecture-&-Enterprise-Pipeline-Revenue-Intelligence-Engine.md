## Integration Tips

- **Salesforce/HubSpot offline conversion import**: Use Microsoft Advertising's conversion import API (or Zapier connector) to push `Opportunity Created` and `Closed Won` events back into the platform daily — this unlocks Target CPA and ROAS bidding based on real pipeline data, not just form fills
- **LinkedIn Campaign Manager sync**: Run a shadow comparison — target the same LinkedIn audiences via both LinkedIn Ads and Microsoft Ads LinkedIn Profile Targeting, then compare CPL and pipeline quality. Many B2B teams find MSAN delivers 40–60% lower CPL for equivalent audiences
- **Google Ads keyword parity**: Import your top Google search campaigns into Microsoft Ads via the Microsoft Import tool (Campaigns → Import → Import from Google Ads), then customize bids and LinkedIn overlays. Saves 80% of build time.
- **Clearbit/6sense integration**: If using intent data platforms, create custom audience lists from in-market accounts and upload to Microsoft Ads as Customer Match — target accounts showing buying signals across both search and display
- **Looker Studio / Power BI reporting**: Connect Microsoft Advertising API to your BI layer alongside Google Ads and LinkedIn Ads for unified paid media performance reporting; map CPL and pipeline contribution across all three channels in a single executive dashboard

## Troubleshooting

**Problem: Microsoft Ads CPCs are actually higher than Google for the same keywords**
Solution: LinkedIn Profile Targeting bid adjustments are additive and can over-inflate costs for popular ICP segments. Audit your bid adjustment stack — if you have +40% for C-Suite AND +35% for HR job function AND +25% for target industry, the combined multiplier can be 2.4x your base bid. Cap total bid adjustments at +80% maximum, or switch LinkedIn targeting from "bid adjustment" to "target and bid" to only show ads to matched profiles (reduces volume but improves efficiency).

**Problem: MSAN click-through rates are below 0.1% and CPMs are high**
Solution: Microsoft Audience Network performance is heavily placement-dependent. Run a placement report and exclude underperforming placements (often gaming or entertainment sites). Enable "Similar audiences" from your existing converters to improve relevance scoring. Ensure image assets meet MSAN specs (1200×628 for wide, 1200×1200 for square) — blurry or text-heavy images significantly underperform. Native ads should look editorial, not promotional.

**Problem: LinkedIn Profile Targeting match rate is below 30%**
Solution: LinkedIn matching requires users to be logged into LinkedIn when browsing Bing or MSAN placements. Match rates improve with: (a) broader LinkedIn targeting criteria (avoid combining 4+ LinkedIn dimensions simultaneously), (b) adding company name targeting alongside job title (company match is higher than job title match), (c) uploading CRM contact lists as Customer Match and using them as LinkedIn lookalike seeds. A 30–50% match rate is normal — don't optimize toward 100%, as unmatched traffic often still converts at acceptable rates and provides reach beyond your known ICP.
