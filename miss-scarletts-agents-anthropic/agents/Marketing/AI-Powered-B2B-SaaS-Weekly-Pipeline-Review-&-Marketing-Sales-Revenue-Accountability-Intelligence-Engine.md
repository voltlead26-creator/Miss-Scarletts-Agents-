# AI-Powered B2B SaaS Weekly Pipeline Review & Marketing-Sales Revenue Accountability Intelligence Engine

**Difficulty:** Advanced | **Time:** 20-30 min | **Tags:** b2b, saas, pipeline, revenue, sales-alignment, demand-gen, automation

## Overview
Transforms raw CRM pipeline data into a comprehensive weekly pipeline review package — covering coverage analysis, marketing attribution, at-risk deal identification, and marketing action plans — so revenue teams can run data-driven weekly sales-marketing alignment calls without manual spreadsheet wrangling. Use it every Monday before your pipeline review call or before any QBR.

**5. MARKETING ACTION PLAN (next 7 days)**
Specific, assignable tasks with owners, deadlines, and expected pipeline impact.

**6. CALL TALKING POINTS**
3-5 bullet points the marketing leader can use to open the pipeline review meeting.

## Integration Tips

- **Salesforce/HubSpot CRM**: Run a pipeline report filtered by stage, source, and age; paste the CSV summary or key metrics directly into the prompt. Use the action plan output to create tasks/activities directly in CRM for AE follow-up.
- **Clari or Gong Forecast**: Export the AI-generated stage-weighted forecast and compare against Clari's call — divergence above 15% signals data quality issues worth investigating before the call.
- **Google Sheets / Notion**: Paste the tables directly — both tools accept markdown tables natively. Build a weekly pipeline dashboard by running this prompt every Monday and archiving the scorecard tab.
- **Slack**: Paste the "Call Talking Points" section into the #revenue or #pipeline Slack channel as a Monday morning standup note; it creates automatic alignment before everyone joins the call.
- **Marketo/HubSpot Marketing Hub**: Use the "30-Day Marketing Program Recommendations" section to build or modify campaign workflows. Program pauses and budget reallocations identified by the prompt can be executed directly in your marketing automation platform.
- **Zapier/Make**: Automate the pipeline data extraction → prompt submission → Notion/Slack delivery on a Monday 7am schedule using a Zapier sequence: CRM report → webhook → Claude API → Notion page create → Slack message.
- **Tableau/Looker**: Feed the prompt's attribution and program ROI output into your BI dashboard as structured weekly annotations to track trend lines over time.

## Troubleshooting

**Problem:** Pipeline coverage looks artificially high but quota attainment is consistently low.
**Solution:** Add the slip rate adjustment to the stage-weighted calculation (multiply each stage's weighted pipeline by 1 minus the historical slip rate). A Stage 4 deal with 60% win probability and a 25% slip rate to next quarter is effectively worth 45% of face value this quarter. Ask the prompt to apply slip-rate-adjusted weighting.

**Problem:** Marketing attribution numbers seem too low and sales disputes ownership of most deals.
**Solution:** Use both first-touch and multi-touch attribution models. Run the prompt twice — once with first-touch sourcing data and once with any-touch influence data. The gap between the two numbers is your marketing influence story. Most B2B orgs should see marketing-influenced pipeline at 60-80% even when sourced pipeline is 25-35%.

**Problem:** The action plan output is too generic (e.g., "send a case study") rather than specific.
**Solution:** Provide more deal-specific context: the AE's exact notes on the blocker, the prospect's industry and pain point, and what assets you already have available. The more specific your stuck deal inputs, the more targeted the recommended intervention. Include the prospect's job title and specific objection language if available.
