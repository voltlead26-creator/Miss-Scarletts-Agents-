## Overview

This intelligence engine quantifies the revenue impact of sales enablement investments by correlating content consumption, training completion, and tool adoption with deal velocity, win rates, and quota attainment — producing a CFO-ready enablement ROI model that proves the business case for continued investment. Use it when you need to defend your enablement budget, identify which programs actually drive revenue, and build a data-driven roadmap for enablement prioritization.

## Integration Tips

**Salesforce / HubSpot CRM:**
- Create custom opportunity field: `enablement_content_shared` (boolean) — auto-populated via Highspot native Salesforce sync
- Build report: "Won vs. Lost Opportunities by Content Engagement" segmented by ACV band and competitor
- Schedule monthly data export to Google Sheets for cohort analysis — join on `opportunity_owner_email` as the common key across all systems

**Highspot / Seismic / Showpad:**
- Enable native CRM integration so content sharing events are logged to opportunity records automatically (zero rep effort)
- Export monthly active user report + asset-level engagement metrics (views, shares, time on page per rep)
- Use Highspot Analytics "Deals" view to directly tie content to deal outcomes — this is the cleanest attribution data source

**Gong / Chorus:**
- Pull rep-level monthly summary: talk ratio, questions asked per call, MEDDPICC flag completion rates, call score trends
- Set up Gong Engage tracker: mentions of competitor names → flag deals as competitive → pull win/loss outcomes
- Coaching correlation analysis: export coaching interaction frequency per rep per month → join to win rate → build scatter plot

**Workramp / Lessonly / Mindtickle:**
- Schedule automated weekly export: cert completion status by rep → push to Salesforce custom field via Zapier
- Track: hire date → certification completion date → first opportunity created → first closed-won
- Build automated trigger: rep completes MEDDPICC cert → Slack notification to manager + CRM field update → schedule 30-day post-cert win rate check-in

**Zapier / Make for Automated Reporting:**
- Monthly trigger: pull data from Highspot API + Gong API + Workramp export + SFDC Reports → consolidate in BigQuery or Google Sheets → run analysis prompt → auto-generate report → post to Slack #enablement-intelligence channel

## Troubleshooting

**Problem: Reps don't consistently tag content usage to opportunities, creating attribution gaps in your data**
Solution: Enable native CRM sync in Highspot/Seismic so content sharing events log automatically to opportunity records — no rep action required. For analysis, use the 50-60% of deals where data *is* clean; the statistical signal will still hold. Supplement with a manual audit of 20-30 deals each quarter (call records + email threads) to validate your model against ground truth.

**Problem: CRO or CFO challenges the ROI claims because "correlation doesn't prove causation"**
Solution: Design a prospective controlled experiment rather than a retrospective observational study. Randomly assign a cohort of 12-15 reps to a structured enablement intervention (specific content package + coaching cadence) for one quarter, while a matched control group continues business-as-usual. Randomization eliminates selection bias. Measure win rate, ACV, and cycle length for both groups. This is the gold standard attribution methodology that withstands CFO scrutiny — and it only requires one quarter of patience.

**Problem: Enablement data is scattered across 4-6 different systems with no unified view, making analysis extremely time-consuming**
Solution: Start with a lightweight Google Sheets model using `rep_email` as the universal join key across all system exports. Map: (rep email → quota attainment from SFDC) + (rep email → Highspot MAU from platform export) + (rep email → certification status from LMS export) + (rep email → Gong call score from Gong export). This produces a "rep-level enablement health scorecard" in under 3 hours. For a scalable solution, pipe all sources into a warehouse (Snowflake/BigQuery) and build a Looker/Tableau dashboard — typically a 2-3 week RevOps project with a 2-year payback.
