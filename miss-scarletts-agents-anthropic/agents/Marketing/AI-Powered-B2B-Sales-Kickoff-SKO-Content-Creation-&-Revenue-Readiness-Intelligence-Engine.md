## Integration Tips

- **Notion / Confluence**: Export the full SKO agenda as a database with session owners, status, and file attachments; use Notion AI to generate speaker prep briefs from session outlines.
- **Gong / Chorus**: After SKO, create a call library tagged by SKO skill (e.g., "multi-threading," "competitive displacement") so managers can share real examples during reinforcement coaching.
- **Salesforce / HubSpot**: Add a custom "SKO Play Used" field to opportunity records so you can measure which new plays are generating pipeline; create a 90-day post-SKO dashboard.
- **Highspot / Seismic / Guru**: Upload all battle card refreshes, product update one-pagers, and competitive sheets with version control; tag by SKO year so reps always find current material.
- **Slack / Teams**: Set up a dedicated #sko-reinforcement channel where the SKO theme is kept alive — weekly win stories, manager coaching prompts, and leaderboard updates for the 90-day competition.
- **Zoom / Hopin**: For hybrid SKOs, use AI-generated session summaries immediately after each session so remote attendees can catch up without watching full recordings; create auto-generated Q&A digests.
- **Lattice / Leapsome**: Connect SKO skill certifications to individual performance plans so post-SKO coaching is formally tracked against development goals.

## Troubleshooting

**Problem: The generated agenda feels generic and doesn't reflect our specific product or team challenges.**
Solution: Front-load the prompt with more specific performance data — actual win/loss themes from your CRM, verbatim quotes from rep exit interviews, specific competitor claims you're seeing in the field. The more specific your "last year's performance" and "skill gaps" inputs, the more tailored the output. Run the competitive section separately with a deep competitor brief pasted into context.

**Problem: Training session curricula are too conceptual and lack practical exercises.**
Solution: After generating the curriculum outline, run a follow-up prompt: "Now design the role-play scenario for Session B in full detail — write the prospect persona card, the opening situation, the 3 curveball objections the prospect throws, and the scoring rubric judges use." Ask for specifics on one session at a time for maximum depth.

**Problem: The CEO/CRO keynote outline doesn't match how our executive actually communicates.**
Solution: Feed 2-3 examples of prior executive communications (all-hands transcripts, board deck narratives, or recorded QBR remarks) as style context before generating the outline. Add a style note like: "Our CEO is data-led and uses the structure: problem → insight → implication → action. She avoids emotional appeals and prefers confident understatement." The AI will calibrate accordingly.
