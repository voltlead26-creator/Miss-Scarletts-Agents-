## Integration Tips

**Salesforce:** Paste battlecard OUTPUT 1 into the Opportunity Notes or create a custom "Competitive Deal Playbook" field. Use the Deal Acceleration Actions as Tasks with due dates assigned to the AE.

**HubSpot CRM:** Store battlecard outputs as Deal Properties or create a "Competitive Intelligence" note type. Trigger this prompt via workflow automation when deal stage moves to "Evaluation" and a competitor is logged.

**Gong / Chorus:** After each competitive call, feed the call transcript into the Advanced Customizable Version to update objection handling scripts based on what actually came up — closes the loop from deal intelligence back to battlecard library.

**Notion / Confluence:** Build a Battlecard Library page per competitor. Use this prompt monthly to refresh with new intelligence signals (G2 reviews, product releases, pricing changes, layoffs, funding news).

**Slack (Sales Channels):** Connect to a #competitive-intel Slack channel. When a rep logs a competitor in Salesforce/HubSpot, auto-trigger this prompt and post the one-page battlecard summary directly to Slack as a deal-specific alert.

**Clay:** Use Clay to pull real-time data on the competitor (LinkedIn headcount changes, job postings, G2 review sentiment) and pre-populate the Competitor Profile section of the Advanced Prompt before running it — creates a fully automated competitive intelligence pipeline.

**Zapier / Make:** Trigger this prompt automatically when: (1) a new competitor is added to a deal in CRM, (2) a deal moves to late-stage, or (3) a rep requests a battlecard via a Slack slash command.

## Troubleshooting

**Problem: Objection scripts sound too salesy or scripted**
Solution: Add this constraint to the prompt: "Write each response as if a trusted friend who happens to sell this product is helping another friend navigate a decision — direct, honest, slightly self-deprecating where warranted." Remove any language that sounds like a brochure.

**Problem: Business case paragraph is too generic without company-specific data**
Solution: Pull 3-5 specific data points about the prospect before running (recent earnings mentions, job posting signals, LinkedIn headcount growth, Glassdoor review themes) and add them to the Deal Context section. The more specific the input, the more compelling the output.

**Problem: Trap-setting questions feel too obvious or aggressive in practice**
Solution: Add "Frame each question as genuine curiosity from someone trying to help them make the right decision, not uncover a weakness. The rep should be able to ask this of a champion who likes both vendors." This softens tone without losing strategic intent.
