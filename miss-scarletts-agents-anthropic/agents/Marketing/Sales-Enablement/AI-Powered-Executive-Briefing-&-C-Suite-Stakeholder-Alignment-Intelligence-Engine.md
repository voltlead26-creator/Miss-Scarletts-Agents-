## Overview
This engine generates fully personalized executive briefing packages — tailored to each C-suite stakeholder's risk profile, financial language, and strategic priorities — so your champion can walk into the boardroom armed with the exact narrative that gets the deal approved. Use it on any enterprise deal where budget approval requires executive sign-off beyond your day-to-day contact.

## Troubleshooting

**Problem: Champion says the financial model numbers "seem too high" and won't use the brief**
Solution: The champion is protecting their credibility — they don't want to present numbers to the CFO that get challenged. Run the model again with more conservative assumptions (haircut savings by 40%, extend payback period, remove all soft savings). A credible conservative case is worth more than an aggressive optimistic one. Also provide the champion with the explicit methodology behind each number so they can defend it if pressed.

**Problem: The CTO brief isn't landing — getting pushback on security claims the vendor hasn't formally confirmed**
Solution: Never include a security certification or compliance claim unless you have a current third-party audit report to back it. Replace unconfirmed claims with "pending validation" language and flag them for the vendor's security team to confirm in writing within 24 hours. One false security claim can kill a deal and damage your champion's reputation permanently.

**Problem: Different executives are receiving conflicting versions of the value story**
Solution: Anchor all stakeholder briefs to a single shared "source of truth" financial model. Each executive brief pulls from the same base numbers — the CFO, CTO, and CEO all see the same investment figure and the same 3-year value. Variation between briefs should only be in *framing* (financial lens vs. technical lens vs. strategic lens), never in underlying facts. Use the Stakeholder Intelligence Matrix to track which narrative each executive received.
