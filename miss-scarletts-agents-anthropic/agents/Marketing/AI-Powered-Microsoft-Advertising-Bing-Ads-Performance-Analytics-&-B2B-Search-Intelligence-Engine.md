## Troubleshooting

**Issue 1: LinkedIn Profile Targeting shows minimal impressions despite active settings**
*Cause:* Microsoft requires a minimum audience size to activate LinkedIn Profile Targeting — typically 1,000+ matched users. Very narrow combinations (e.g., specific company + specific job title + specific seniority) may fall below threshold.
*Fix:* Broaden targeting to 2-3 dimensions instead of 4-5. Use industry + job function + seniority as your core combination before adding company size or company name. Check audience size estimate in the audience targeting panel before saving.

**Issue 2: UET tag reporting zero conversions despite form submissions occurring**
*Cause:* Most common causes: (1) Tag fires on all pages but event tag not implemented on confirmation/thank-you page; (2) Single-page application (SPA) not triggering page view events on URL change; (3) Tag blocked by privacy consent management platform (CMP).
*Fix:* Use Microsoft's UET Tag Helper Chrome extension to debug tag firing in real time. For SPAs, implement event-based conversion tracking instead of page-load based. Ensure UET fires post-consent for GDPR-compliant implementations.

**Issue 3: Performance Max campaign consuming entire budget with poor lead quality**
*Cause:* Performance Max optimizes for conversion volume — if your primary conversion action is a top-of-funnel content download, PMax will generate high volume but poor quality leads. PMax also has limited audience exclusion capabilities vs. Search.
*Fix:* Set primary conversion action to demo request or bottom-funnel form submission. Add high-quality audience signals (remarketing lists of past converters, customer match of closed-won customers) to guide PMax optimization. Set a conversion value multiplier: demo request = $500 signal value, content download = $25 signal value. If PMax continues to underperform, pause it and run dedicated Search campaigns for your core intent terms.
