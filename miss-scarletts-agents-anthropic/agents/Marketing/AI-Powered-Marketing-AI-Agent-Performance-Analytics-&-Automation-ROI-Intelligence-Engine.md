## Integration Tips

- **Salesforce / HubSpot:** Pull AI tool attribution data by creating a custom field "AI-Assisted" on campaigns and contacts; filter pipeline reports by this field to isolate AI-influenced revenue
- **Looker / Tableau:** Build a live AI Performance Dashboard with one tile per agent tracking composite score over time — set up weekly automated refresh so the scorecard is always current
- **Notion / Confluence:** Export the ROI report as the source document for your annual AI budget justification; link to it from your marketing planning wiki so it's auditable
- **Slack:** Set up a weekly AI performance digest using a Zapier workflow — pull top-line metrics from your BI tool and post to your marketing-ops channel every Monday
- **Marketo / Pardot:** Tag all AI-generated or AI-personalized email variants with a UTM parameter (`utm_content=ai-personalized`) to enable clean A/B comparison against human-authored variants
- **Clay / Apollo / ZoomInfo:** Measure enrichment accuracy by sampling 100 records monthly and manually verifying 5 fields — create a data quality KPI dashboard to track over time

## Troubleshooting

**Problem: Can't isolate AI agent impact from other changes happening at the same time (new SDRs hired, ICP shifted, seasonality)**
Solution: Build a "synthetic control" — find the metric trend for the 6 months before AI deployment, project it forward at the same rate, and compare actual performance to that projected baseline. Flag confounds explicitly in the report. Even directional evidence with honest caveats is more credible than a false precision number.

**Problem: Sales team dismisses AI attribution claims ("we'd have closed those deals anyway")**
Solution: Run a holdout experiment — for 30 days, deliberately not use the AI agent for 20% of your pipeline (randomly selected) and compare conversion rates. If you can't run a holdout, pull the deals AI didn't touch in the last 90 days and compare close rates. Let the data make the argument, not the marketing team.

**Problem: The ROI calculation shows low or negative return on one tool but the champion on the team refuses to consider retiring it**
Solution: Separate the analysis from the decision. Present the scorecard as data ("this is what we found") not as verdict ("therefore we cut it"). Ask the champion to describe what the tool would need to show in the next 60 days to prove its value — document their answer as the success criteria, then hold the 60-day review. Either the tool improves or the champion owns the decision.
