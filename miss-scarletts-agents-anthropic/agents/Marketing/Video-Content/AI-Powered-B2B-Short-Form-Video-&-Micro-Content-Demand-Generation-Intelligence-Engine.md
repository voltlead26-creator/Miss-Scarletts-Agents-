## Integration Tips

**OpusClip / Vidyo.ai (AI clip extraction):**
- Connect your YouTube channel or upload MP4 files directly
- Use the "B2B/Tech" content category for better clip scoring
- Set minimum clip length to 30 seconds to avoid micro-fragments
- Export directly to vertical (9:16) format for Shorts and TikTok

**Descript (AI editing + captions):**
- Use "Overdub" to cleanly fill verbal filler (ums, ahs) without re-recording
- Auto-captions → export as SRT and import to LinkedIn/YouTube natively for better algorithm performance
- Use "Underlord" AI to automatically remove silences and tighten pacing

**HubSpot (attribution):**
- Create a custom property "Video Content Influenced" on Contact/Deal records
- Build a workflow: If UTM source = "linkedin-video" or "youtube-shorts" → tag contact + notify assigned rep
- Use Workflows to auto-enroll video viewers (from LinkedIn matched audiences) into nurture sequences

**Buffer / Hootsuite (scheduling):**
- Enable "Best Time to Post" AI feature — for B2B LinkedIn, this typically surfaces Tue-Thu 7-9am
- Use "Drafts" to batch-load entire month's video calendar in one session
- Set up auto-tagging with UTM parameters on all bio links

**Zapier automation sequence:**
1. Trigger: New video uploaded to Google Drive (from Descript export)
2. Action: Rename file with platform + clip number + date
3. Action: Upload to Buffer as a Draft
4. Action: Create HubSpot campaign record and link video asset
5. Action: Post Slack notification to marketing team for final review

**Warmly / RB2B (anonymous visitor de-anonymization):**
- Install pixel on landing pages linked from video bio links
- When an anonymous visitor hits the page, Warmly identifies their company via IP
- Trigger: Alert assigned AE when a target account visits after video engagement
- This closes the "dark social" attribution gap for B2B video

## Troubleshooting

**Problem: AI-clipped videos feel choppy or cut mid-thought**
Solution: Before running AI clip extraction, export a "cleaned" version of your long-form video through Descript — remove filler words, tighten gaps between sentences. AI clip tools perform 40-60% better on clean source audio. Also increase minimum clip length setting from 15 seconds to 30 seconds minimum.

**Problem: LinkedIn video views are high but no pipeline influence detected**
Solution: Audit your CTA architecture. If every clip ends with "follow for more tips," you're building an audience, not a pipeline. Implement a 3:1 ratio — 3 pure value clips, then 1 soft CTA clip (demo invite, assessment, report download). Also verify your UTM parameters are firing correctly on bio links using LinkedIn's built-in URL tracking.

**Problem: Clip production is taking more than 4 hours per 20 clips**
Solution: The bottleneck is almost always clip selection (human review step). Build a scoring rubric so reviewers can approve/reject in under 60 seconds per clip: Rate each AI-suggested clip on a 1-3 scale for (a) hook potential, (b) ICP relevance, (c) standalone clarity. Auto-approve anything scoring 7+ out of 9. Create a Notion template with the rubric embedded so any team member can do the review.
