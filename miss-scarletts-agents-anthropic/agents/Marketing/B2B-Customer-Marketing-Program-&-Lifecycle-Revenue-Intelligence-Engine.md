## Integration Tips

**Gainsight / ChurnZero / Totango (Customer Success Platform):**
- Export health score segments daily to HubSpot/Marketo via native integration or Zapier
- Create health score-based smart lists for automated campaign enrollment
- Set up bi-directional sync: when marketing email opens, log activity in CS platform timeline

**HubSpot / Marketo (Marketing Automation):**
- Build lifecycle stage property synced from CS platform health score bands
- Create customer-specific email suppression lists: "Do Not Market — Active Escalation" updated by CS team
- Use HubSpot Sequences for CSM-triggered personal email campaigns (feel 1:1, run at scale)

**Salesforce (CRM):**
- Build "Expansion Opportunity" campaign influence field — track every marketing touch on expansion deals
- Create custom report: NRR by marketing lifecycle stage to prove program ROI to CRO
- Integrate Pendo/Mixpanel product usage data as custom fields on Account object for campaign triggers

**Intercom / Pendo (In-App Messaging):**
- Layer in-app messages on top of email sequences for high-value feature adoption moments
- Use in-app NPS (score collected in product) as real-time advocacy signal
- Trigger "expansion intent" flag when user visits pricing/upgrade page 2+ times in 30 days

**Notion / Confluence (Internal Documentation):**
- Build a "Customer Story Library" with intake form, production workflow, and approval status tracker
- Create the "CS-Marketing Playbook" as a shared document updated monthly

**Zapier / Make (Workflow Automation):**
- Health score drops below threshold → Slack alert to CSM + enrollment pause on active marketing campaigns
- NPS score of 9-10 submitted → add to advocacy outreach sequence + notify Customer Marketing manager
- Contract renewal date 90 days out → trigger "Renewal Intelligence" campaign sequence in HubSpot

## Troubleshooting

**Problem: CSMs complain that marketing is spamming their customers, damaging relationships.**
Fix: Implement a "marketing pause" flag in your CRM that CSMs can toggle for any account. Run a monthly sync where CS reviews all campaigns planned for their accounts. Start with only health score > 70 accounts for any expansion campaigns — never market to red accounts without explicit CSM approval. Add an unsubscribe tracking dashboard to your weekly CS/Marketing sync and set a 2% threshold as an automatic audit trigger.

**Problem: Customers ignore lifecycle emails — open rates below 15%, zero expansion conversions.**
Fix: Audit your "From" name — campaigns sent from "Revenify Marketing" perform 40-60% worse than campaigns sent from a named person ("Sarah from Revenify Customer Success"). Shift to personalization-first: use merge tags for specific usage data ("You've run 47 forecasts this month — here's what Analytics Pro customers do differently"). A/B test subject lines using customer-specific metrics vs. generic value propositions. If deliverability is the issue, check if customer domains are on suppression lists from previous bulk sends.

**Problem: Can't get customers to agree to case studies or references despite being very happy (NPS 9-10).**
Fix: The #1 barrier is perceived risk, not willingness. Remove friction: offer a "behind-closed-doors reference call only" option (no public content required). Create a reference "ladder" — start with anonymous data contribution ("we'll cite your results without naming you"), then progress to named case study. Offer reciprocal value: invite reference customers to your executive advisory board, offer free conference passes, or provide early access to product roadmap. Personalize the ask: a CEO-to-CEO email request converts 3-4x better than a CSM request. Build a "reference playbook" for CSMs with email templates, FAQ objection handlers, and escalation paths.
