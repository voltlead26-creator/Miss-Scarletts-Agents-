**B. PMax-Search Campaign Conflict Prevention**

The #1 mistake in B2B PMax: it cannibalizes your best-performing Search campaigns.

Prevention tactics:
- Create Brand Exclusion lists in PMax — exclude all branded search terms so PMax doesn't serve on brand queries (those should stay in your brand Search campaigns with better control)
- Set PMax to 5–10% of total budget initially; expand only if PMax pipeline quality matches Search pipeline quality
- Monitor: In Google Ads search term reports, filter by campaign = PMax and review every 2 weeks for irrelevant queries
- Campaign priority: Search campaigns take priority over PMax when same user matches both — ensure Search campaigns are always running

**A. Bidding Strategy Phases**

B2B companies consistently destroy Google Ads ROI by activating Smart Bidding too early or with wrong conversion signals. Follow this phased approach:

Phase 1 (Days 0–45): Manual CPC with Enhanced CPC
- Purpose: Accumulate conversion data without algorithmic distortion
- Target: 30+ conversions per campaign before enabling Smart Bidding
- Action: Set Max CPC bids based on [ACV × Lead-to-Close Rate × Target ROAS] ÷ Expected Conversion Rate
  - Example: $72K ACV × 12% lead-to-close × 3× ROAS target ÷ 5% expected CR = $518 max CPC
  - Apply a haircut (60–70% of max CPC) while learning

Phase 2 (Days 45–120): Target CPA per Campaign
- Activate once each campaign has 30+ conversions in a 30-day window
- Set initial Target CPA = current average CPA × 1.2 (give Smart Bidding room to learn)
- Target CPA calculation: [Pipeline Influenced per Lead × Target Marketing Efficiency Ratio]
  - Example: $72K ACV × 38% lead-to-SQL × 31% SQL-to-close = $8,481 pipeline per lead
  - If target marketing efficiency = 15:1 pipeline-to-spend, Target CPA = $8,481 ÷ 15 = $565
- Reduce Target CPA by 10% every 2 weeks if performance holds

Phase 3 (Days 120+): Target ROAS with Offline Conversion Import
- Requires: CRM offline conversion import configured and working (see Module 6)
- Google Ads can now optimize toward actual pipeline and closed revenue, not just form fills
- Set Target ROAS = [Pipeline Generated per $ Spent Goal]
  - Example: If goal is $12 pipeline per $1 spent, Target ROAS = 1,200%
- This is the highest-performing configuration for B2B — most companies never reach it

- Reduce bids by 30–50% on weekends and after 9 PM on weekdays
- Use Ad Scheduling to eliminate spend outside business hours if budget is constrained

**A. Offline Conversion Import Setup**

This is the single highest-impact configuration for B2B Google Ads that 90% of teams skip:

Step 1: Capture Google Click ID (GCLID) in CRM
- Add a hidden GCLID field to every Google Ads landing page form
- Auto-populate via URL parameter: `?gclid={gclid}`
- Store GCLID in Salesforce/HubSpot lead/contact record at point of form submission
- Verify: After 48 hours of running, confirm GCLID is populating in CRM records from Google Ads traffic

Step 2: Define Offline Conversion Events to Import
- Conversion 1: MQL — when lead status = Marketing Qualified (import within 24 hours)
- Conversion 2: SQL/Opportunity Created — when opportunity stage = Discovery/Demo Booked
- Conversion 3: Pipeline — when opportunity reaches Stage 2+ with assigned dollar value (import as conversion value = opportunity amount)
- Conversion 4: Closed Won — when opportunity stage = Closed Won (import as conversion value = ACV)

Step 3: Automated Import Mechanism
- Salesforce: Use Zapier, native Google Ads connector, or Salesforce Data Export with Google Ads API
- HubSpot: Use native Google Ads integration (Settings → Integrations → Google Ads → Offline Conversion Sync)
- Upload frequency: Daily minimum; hourly preferred for fast-moving pipelines
- Attribution window: Use 90-day click-through window for B2B (matches average sales cycle)

Step 4: Validate Attribution in Google Ads
- Navigate to: Conversions → All conversions → filter by "Imported from Salesforce/HubSpot"
- Verify conversion volume matches what you see in CRM (allow 48-hour delay)
- Check: Google Ads campaigns should start showing "Pipeline Value" and "Revenue" columns once data accumulates

## Troubleshooting

**Problem: Google Ads spend is being exhausted by noon, missing afternoon traffic**
Solution: Check impression share lost due to budget (Campaigns view → Columns → Competitive Metrics). If > 30% impression share is lost to budget, either (a) increase budget for the affected campaigns, (b) apply ad scheduling to limit spend to peak conversion hours (typically 9 AM–5 PM in target timezone), or (c) reduce bids on low-converting times using bid adjustments. Do NOT simply pause campaigns mid-day — Google penalizes inconsistent delivery and it resets Smart Bidding learning.

**Problem: Offline conversions aren't appearing in Google Ads after 2 weeks**
Solution: Run through this diagnostic checklist: (1) Confirm GCLID is being captured in CRM — go to 5 recent Google Ads leads and check if GCLID field is populated; (2) Check that GCLID is being included in your offline conversion upload file (column name must be exactly "Google Click ID"); (3) Verify that the conversion action in Google Ads is set to "Imported" type (not "Website" type); (4) Confirm conversion upload is using the correct timezone matching your Google Ads account timezone; (5) Check the "Diagnostics" tab in your Google Ads conversion action — it will show specific error messages if uploads are failing.

**Problem: Performance Max is cannibalizing best-performing Search campaigns (impressions and clicks dropping on Search campaigns)**
Solution: This is a common and serious issue. Fix: (1) In PMax campaign → Settings → Brand exclusions — add ALL your brand terms so PMax never serves on branded queries; (2) Verify your Search campaigns have the same or higher campaign priority settings; (3) Create a "PMax Exclusion" negative keyword list — add your top 20 exact-match category keywords as negatives within PMax (you cannot add traditional negative keywords to PMax, but you can exclude placements and use the "brand exclusion" feature for search terms); (4) If cannibalization continues, reduce PMax budget to 2–3% and monitor Search campaign impression share recovery over 7 days.
