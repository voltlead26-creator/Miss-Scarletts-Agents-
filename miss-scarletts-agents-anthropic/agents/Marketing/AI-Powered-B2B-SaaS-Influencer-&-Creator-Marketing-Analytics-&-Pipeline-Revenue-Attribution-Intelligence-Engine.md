## Success Metrics

- **Pipeline-influenced ROAS per creator** — primary metric; B2B benchmark: >5x is strong, >20x is exceptional for well-tracked programs
- **Cost-per-influenced-SQL** — more actionable than CPL; B2B SaaS target: $1,000-$3,500 depending on ACV
- **ICP audience alignment %** — for LinkedIn/YouTube creators, target >50% VP+ or Director-level buyers matching your ICP industry and company size
- **Attribution coverage rate** — % of influenced pipeline that has trackable attribution (UTM, promo code, or self-reported); target >60%
- **Creator content engagement quality** — engagements-from-ICP-accounts ÷ total engagements; use LinkedIn Analytics or third-party tools to segment
- **Dark social triangulation accuracy** — validate by comparing self-reported survey data to UTM-tracked attribution; if self-reported is >3x tracked, attribution infrastructure needs investment
- **Roster renewal rate** — % of Tier 1/2 creators renewed each quarter; high churn in top performers signals relationship management issues

## Integration Tips

- **HubSpot:** Create a custom "Influencer Source" property on Contacts. Set up UTM parameters with `utm_source=influencer&utm_medium=[platform]&utm_campaign=[creator-name]`. Build a HubSpot "Campaigns" object for each creator partnership to track influenced contacts and pipeline using Campaign Influence reporting.
- **Salesforce:** Use Campaign Influence with Primary Campaign Source and add custom Influence fields for influencer touchpoints. Set up Lead Source = "Creator/Influencer" and add a Creator Name picklist field. Connect to Tableau or Looker for pipeline-by-creator dashboards.
- **Supermetrics or Funnel.io:** Pull creator-specific UTM data from Google Analytics 4 alongside CRM pipeline data into a unified dashboard. Schedule weekly auto-exports to Google Sheets to feed this prompt's analysis template.
- **LinkedIn Analytics:** Export Creator Analytics from LinkedIn Campaign Manager (if running Thought Leader Ads) or use the LinkedIn Page Analytics API for organic creator mention tracking. Cross-reference follower demographics against your ICP criteria.
- **Promo Code Tracking:** For podcast sponsorships without clickable links, assign unique vanity URLs (e.g., yourproduct.com/podcast-name) that redirect with UTM parameters. Track promo code redemptions in your CRM as a separate lead source with the podcast name as Campaign.
- **Survey Integration:** Add "How did you hear about us?" to your demo request form with a multi-select dropdown including creator names. Feed self-reported data into a Notion database or Google Sheet and reconcile with UTM-tracked attribution monthly.
- **Zapier Workflow:** Trigger this analysis prompt monthly: pull UTM data from GA4 via Google Sheets, pull pipeline data from HubSpot via API, merge into the template, run through Claude API, auto-post executive summary to #creator-marketing Slack channel.

## Troubleshooting

- **Problem:** Most influencer-driven pipeline is untracked because buyers don't click links — they search directly after seeing creator content.
  **Solution:** Implement the dark social triangulation method in Section 4. Add "How did you hear about us?" to all demo request forms and quarterly to your customer base. Monitor branded search volume in Google Search Console weekly and correlate spikes to creator content publish dates. Even 30-40% attribution coverage is sufficient to directionally prove ROI — focus on building the full tracking stack over 2-3 quarters rather than waiting for perfect data before proving value.

- **Problem:** The creator with the largest audience (500K+ followers) has terrible pipeline ROAS, but the partnership is a "strategic" relationship the CEO wants to maintain.
  **Solution:** Separate brand value from demand generation value in your reporting. Run a "brand impact analysis" for strategic creators — measure share of voice increase, LinkedIn profile view lift for executives, and inbound brand mentions from that creator's audience. Report these as brand KPIs separately from pipeline KPIs. Recommend a lower-cost, lower-commitment format (e.g., one LinkedIn post mention vs. full sponsored series) for strategic relationships with low pipeline ROAS. This preserves the relationship while redirecting budget to high-ROAS creators.

- **Problem:** Creator performance data is scattered across 6 different tools (LinkedIn Analytics, Spotify for Podcasters, YouTube Studio, GA4, HubSpot, spreadsheets), and pulling it together takes 3 hours every month.
  **Solution:** Build a standardized "Creator Performance Template" in Google Sheets with one tab per creator. Assign each creator a unique UTM campaign tag (e.g., `utm_campaign=jordan-merritt-q1-2026`). Set up a monthly 30-minute sync with each creator to get platform-native metrics (impressions, engagements) that you can't pull directly. Use Zapier or Make to auto-pull GA4 UTM data into the sheet weekly. This upfront 4-hour setup reduces monthly reporting time to 45 minutes and makes this prompt's analysis immediately actionable.
