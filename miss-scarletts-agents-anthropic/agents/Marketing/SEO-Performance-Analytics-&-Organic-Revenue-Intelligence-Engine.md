## Overview

Use it when you need to prove organic ROI to leadership, audit an existing SEO program, or build a data-driven organic growth strategy.

**1. ORGANIC REVENUE ATTRIBUTION MODEL**

  Estimated Page Revenue = Monthly Organic Sessions × Organic CVR × ACV × Attribution Weight

**2. KEYWORD OPPORTUNITY SCORING (KOS) MATRIX**
Score every keyword across 4 dimensions (1-10 each):
  - Search Intent Alignment (commercial/transactional = 10, informational = 5, navigational = 2)
  - Revenue Proximity (decision-stage = 10, awareness-stage = 3)
  - Ranking Potential (gap between current rank and achievable rank within 90 days)
  - Business Fit (ICP relevance and conversion probability)
KOS = (Intent × 0.35) + (Revenue Proximity × 0.35) + (Ranking Potential × 0.20) + (Business Fit × 0.10)
Flag Top 15 by KOS as "Priority Keywords for Next Quarter"

**3. CONTENT DECAY PROTOCOL (CDP)**
Identify content decay patterns using:
  - Traffic velocity: pages losing >15% sessions QoQ
  - SERP displacement: pages dropping from top 3 to positions 4-10
  - Cannibalization signals: multiple URLs ranking for same head term
For each decayed page, categorize root cause:
  - Algorithm update victim
  - Competitor outranked
  - Content staleness (outdated data/year references)
  - Technical regression (Core Web Vitals, mobile UX)
  - Internal link dilution
Prescribe specific remediation with effort estimate

**4. SHARE OF VOICE (SOV) COMPETITIVE MODEL**
For your top 30 category keywords:
  SOV = (Sum of your estimated clicks) / (Total search volume × CTR curve) × 100
Compare vs. each named competitor
Identify: Where you're winning, where you're losing, where there's a vacuum to claim

**5. TOPICAL AUTHORITY GAP ANALYSIS**
Using the hub-and-spoke / topic cluster model:
  - Identify your core pillar topics (what you want to own)
  - Map existing content coverage per topic
  - Flag topics with strong intent but thin/no coverage
  - Prioritize clusters with competitor vulnerability (they rank but with low-quality content)

**6. TECHNICAL SEO REVENUE IMPACT CALCULATOR**

  - Affected pages/sessions
  - Estimated traffic recovery if fixed (use industry benchmarks: Core Web Vitals fix → avg +8-12% organic CTR)
  - Revenue upside = recovered sessions × existing CVR × ACV

## Integration Tips

**Google Search Console + Looker Studio:**
Export GSC data (queries, pages, clicks, impressions, position) as CSV. Feed into Module 2's KOS matrix. Build a Looker Studio dashboard with the Priority Keywords Matrix as a live-updating table using GSC connector.

**HubSpot CRM Attribution:**
In HubSpot, set "Original Source" filter to "Organic Search" and filter contacts by lifecycle stage. Export pipeline value to calculate organic pipeline contribution for Module 1. Map to the Revenue Attribution table using UTM-tagged URLs.

**Semrush / Ahrefs API Automation:**
Use Semrush's Position Tracking API to automate the SOV calculation in Module 4. Set up weekly automated exports via Zapier → Google Sheets to track KOS keyword movement. Alert when any Priority Keyword drops more than 3 positions.

**Notion Executive Reporting:**
Paste Module 6 (CMO Executive Briefing) directly into a Notion page. Use Notion's table blocks for the 90-Day Roadmap. Connect to Slack via Notion automation to push weekly organic performance updates to #marketing-leadership channel.

**GA4 + BigQuery:**
For advanced attribution (Module 1), export GA4 event data to BigQuery. Build a custom SQL query joining organic landing page sessions with conversion events and CRM deal values to produce revenue attribution with statistical confidence intervals rather than estimates.

## Troubleshooting

**Problem: "I don't have conversion data broken out by organic channel — everything is lumped together."**
Fix: In GA4, create a custom segment for "Session source = google / organic" and apply to your conversion events. In HubSpot, use the "Original Source Drill-Down 1" property filtered to "Organic Search" to extract organic-sourced contacts and their pipeline value. Even rough segmentation (80% accuracy) produces a usable Module 1 output.

**Problem: "The revenue numbers look way too high / too low — attribution seems off."**
Fix: Adjust the Attribution Weight in Module 1. Blog posts typically drive 0.1-0.3 attribution weight (they influence but rarely convert directly). Solution pages, pricing, and compare pages earn 0.8-1.0. Cross-reference with your CRM's first-touch vs. last-touch attribution reports to calibrate. If the numbers still seem off, use assisted conversion data from GA4 Multi-Channel Funnels report to reality-check.

**Problem: "I only have Google Search Console data — no Semrush or Ahrefs for competitive analysis."**
Fix: Module 4 can be partially executed with free tools. Use Google's "site:competitor.com [your keyword]" to manually audit their content coverage. Use GSC's "Search Analytics" to identify keywords where your position is 4-20 (gap opportunities). For SOV estimation, manually check SERP results for your top 30 keywords and log which competitors appear in positions 1-10 — this manual SOV proxy takes ~2 hours but is 70% as accurate as paid tool data.
