Use this when you're 4–8 weeks into a quarter and need to maximize pipeline from remaining budget without waiting for a full Marketing Mix Model rebuild.

## Success Metrics

Measure weekly: CPL trend, pipeline/$ by channel, pacing rate vs. target.

## Integration Tips

**Salesforce / HubSpot CRM:**
Pull pipeline-by-source reports filtered to "Created Date = This Quarter" and "Stage ≠ Closed Lost." Export as CSV and paste into the Advanced Version's channel performance table. Set a saved report that refreshes weekly so reallocation decisions are based on live data.

**Google Ads / LinkedIn Campaign Manager:**
Pull "Campaign Performance by Week" reports for the current quarter. Track CPL trend week-over-week to identify saturation (CPL rising > 15% over 3 consecutive weeks = saturation signal). Include cost, conversions, and conversion value columns.

**Looker / Tableau / Domo:**
Build a "Budget Pacing vs. Pipeline Pacing" dashboard: two time-series lines showing (a) % of budget spent and (b) % of pipeline target generated, by week. If budget pacing > pipeline pacing by >10 points, trigger reallocation review. Share this dashboard URL in the prompt context for faster AI analysis.

**Zapier / Make Automation:**
Set up a weekly data pull from HubSpot → Google Sheets that auto-populates the channel performance table. Feed the sheet into a scheduled AI analysis run every Monday morning to get weekly reallocation signals without manual data entry.

**Marketo / Pardot:**
Use program performance reports to isolate pipeline influenced by email nurture programs vs. paid acquisition. Separate these in your channel table to avoid double-counting pipeline attribution when making reallocation decisions.

**Google Sheets Budget Tracker:**
Create a "Remaining Budget by Channel" tab that calculates weekly burn rate and projects run-out dates. Formula: `=Remaining_Budget / (Spent_to_Date / Weeks_Elapsed)` gives weeks until budget exhaustion. Feed this into the prompt to ensure reallocation math accounts for spend velocity.

## Troubleshooting

**"The model says reallocate to Google Search but my CPCs have doubled this month."**
Add your current CPC/CPL data explicitly to the prompt. If CPL has risen >25% over 4 weeks, note this as a saturation signal — the engine should reduce the recommended allocation to that channel. Historical pipeline efficiency ($7.88/$) may not reflect current market pricing if the channel has become more competitive.

**"My sales cycle is 90 days but quarter is only 13 weeks. The model says nothing can close this quarter."**
This is often correct — and the honest output is more valuable than a false reallocation plan. In this case, redirect the prompt: instead of net-new pipeline generation, ask it to optimize budget around *advancing existing mid-funnel opportunities* (executive outreach, ROI calculators, deal desk programs) rather than new MQL generation. These have 2–4 week velocity vs. 90-day full cycles.

**"We have three brand campaigns that leadership will never let me pause."**
Add these as "locked spend" in the constraints section. The engine will work around them and only reallocate flexible budget. If locked spend is consuming >50% of remaining budget, note that explicitly — the model should tell you whether the constrained budget is sufficient to close the pipeline gap, or whether you need a leadership conversation about unlocking some of those funds.
