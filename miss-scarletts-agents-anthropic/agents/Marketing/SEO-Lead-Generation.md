# B2B SEO Lead Generation Engine - Organic Search to Pipeline Conversion System

**Difficulty:** Advanced | **Time:** 20-30 min | **Tags:** b2b, seo, lead-generation, content, inbound, demand-gen, strategy

## Overview

This prompt engineers a full B2B SEO lead generation system—from keyword strategy and content architecture to lead capture and pipeline attribution. Use it when you need to build or audit an organic search program that converts buyers actively researching solutions into qualified pipeline.

Make all recommendations specific and implementable. Use real search intent examples, not generic placeholders. Flag any technical SEO prerequisites that must be in place first.

## Integration Tips

- **HubSpot**: Set up `Original Source = Organic Search` smart lists. Create MQL workflows triggered by lead magnet downloads. Use HubSpot Content Strategy tool to track topic cluster performance and organic session attribution on contact records.
- **Salesforce**: Map `Lead Source = Organic Search` and `Lead Source Detail = [page slug]` on form submissions. Build a "Organic Pipeline Influenced" report using campaign influence models. Install Mutiny or Clearbit Reveal to identify high-intent anonymous organic visitors and trigger sales alerts.
- **Google Search Console + Looker Studio**: Connect GSC data to a Looker Studio dashboard. Build a weekly "Ranking Velocity" report showing position changes for your top 50 target keywords. Set up automated email alerts for any top-10 keyword dropping below position 15.
- **Ahrefs / Semrush**: Use the Content Gap tool monthly to find keywords competitors rank for that you don't. Export to a prioritization spreadsheet scoring each gap by: search volume × commercial intent × difficulty inverse.
- **Notion / Airtable**: Build a Content Operations tracker with fields: Keyword, Target URL, Publish Date, Current Position, Monthly Traffic, Leads Generated, Lead Magnet Attached, CRO Test Status. Review weekly.
- **Zapier**: Automate lead magnet delivery: Form submission → create contact in HubSpot → send lead magnet PDF via email → enroll in nurture sequence → slack notify SDR if lead scores above 40.

## Troubleshooting

**Problem: Published articles for 60+ days but seeing zero ranking movement.**
Solution: Check if pages are indexed (GSC Coverage report). If indexed, the issue is likely (a) keyword difficulty too high for your current DR — shift focus to lower-difficulty BOFU terms under 40; (b) content thin or doesn't satisfy search intent — run a content gap analysis against the current page-1 results and add missing sections; (c) no internal links pointing to the article — add internal links from 3–5 existing pages with relevant anchor text.

**Problem: Getting organic traffic but leads are poor quality or not converting.**
Solution: First check alignment between keyword intent and page CTA. A visitor searching "what is OKR software" is not ready to request a demo — they need a TOFU offer (template, checklist). Map your lead magnets to intent stages and replace mismatched CTAs. Second, check if the form is too long — TOFU pages should have 2–3 field forms maximum. Third, review whether your lead magnet actually delivers immediate value or requires a sales follow-up to get value — buyers can tell the difference.

**Problem: Rankings fluctuate wildly week to week, making it hard to measure progress.**
Solution: This is normal in the first 3–6 months (Google "freshness flux"). Focus on 30-day average position, not weekly snapshots. If fluctuations persist past month 6, the likely cause is: (a) thin E-E-A-T signals — add author bio with credentials, cite primary data sources, add expert quotes; (b) page lacks schema markup — add Article, FAQ, or HowTo schema; (c) you're in a YMYL-adjacent category (finance, healthcare, legal tech) where Google holds content to higher standards — invest in authoritative backlinks and expert-reviewed content.
