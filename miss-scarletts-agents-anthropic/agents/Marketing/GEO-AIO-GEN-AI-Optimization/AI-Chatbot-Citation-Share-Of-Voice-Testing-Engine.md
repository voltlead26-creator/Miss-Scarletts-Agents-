## Integration Tips

**Perplexity AI (Manual Testing):**
Run your top 20 queries directly in Perplexity with web search enabled. Screenshot results and record citations in a Google Sheet. Perplexity shows source URLs which makes it easier to identify which specific content assets are being cited. Do this monthly as your baseline measurement.

**ChatGPT with Browsing:**
Test the same query set in ChatGPT with browsing enabled. Note that ChatGPT's citation behavior differs from Perplexity — it's more influenced by training data recency and entity prominence than by real-time web crawling. Discrepancies between platforms reveal whether you have an entity recognition gap vs. a content indexation gap.

**Google AI Overviews:**
Test queries in Google Search and note which appear with AI Overview boxes. Use Google Search Console to identify which of your pages appear in AI Overview source URLs. Export this list monthly and cross-reference with your gap analysis to track closure rate.

**Notion Integration:**
Copy the Citation Probability Matrix and Action Plan tables directly into a Notion database. Add properties for: Owner, Status (Not Started / In Progress / Published / Tracking), Publication Date, 30-Day Lift Achieved. Use Notion automations to send Slack reminders when tracking dates arrive.

**Google Sheets Automation:**
Build a monthly SOV tracker with columns: Date, Query, Platform, Brand Cited (Y/N), Citation Position, Source URL, Notes. Use Apps Script to calculate rolling SOV% and auto-generate monthly trend charts. Share with leadership as your AI search presence dashboard.

**Zapier / Make Workflow:**
Set up a monthly recurring Zap that creates a new Google Sheet tab from your tracker template, sends a Slack reminder to the GEO owner to run queries, and creates a Notion task for the content actions identified in that month's audit.

## Troubleshooting

**Problem: AI citations seem random — same query returns different brands each time**
Fix: AI citation for middle-of-funnel queries has genuine variability, especially in ChatGPT with browsing. Test each query 3 times across 3 sessions and record the most frequent result. For Google AI Overviews, results are more stable — use these as your primary measurement source for SOV tracking. Variability itself is a signal: it means citation authority in this query cluster is contestable, making it a high-priority target.

**Problem: The action plan identifies content to create but we don't know if it's actually getting cited**
Fix: Add a UTM parameter strategy for AI-cited content. Use `?utm_source=ai-search&utm_medium=citation&utm_campaign=[content-slug]` on all new GEO-targeted pages. When Perplexity or ChatGPT browsing mode cites your content, visitors arrive with the full URL including UTM parameters, making attribution trackable in GA4 or HubSpot. Monitor this segment monthly alongside your manual citation testing.

**Problem: We're being cited but always in a negative context (compared unfavorably)**
Fix: This is a sentiment gap, not a citation gap. Run the Advanced Version with explicit instruction to analyze citation sentiment alongside frequency. Then create content that engineers positive citation triggers: specific ROI statistics, named success frameworks, and customer outcome data that AI systems pull when summarizing your brand. The Advanced Version's Phase 2 entity analysis will surface which content assets are creating negative association so you can update or outrank them.
