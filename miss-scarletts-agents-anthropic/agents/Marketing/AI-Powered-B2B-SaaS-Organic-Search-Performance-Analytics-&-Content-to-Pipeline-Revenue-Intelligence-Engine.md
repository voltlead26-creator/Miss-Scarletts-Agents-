## Overview
Analyzes organic search performance across keyword clusters, content assets, and buyer journey stages to attribute pipeline and revenue contribution, identify content ROI gaps, detect ranking decline risks, and generate a prioritized SEO investment action plan. Use this when a CMO demands proof that SEO is generating pipeline, when organic traffic is up but leads are flat, or when you need to decide where to invest content production budget next quarter.

## Success Metrics

- **Pipeline ROAS per content cluster** — primary optimization metric; B2B SaaS benchmark: well-optimized clusters should produce 5-10x content investment in influenced pipeline
- **Cost-per-organic-SQL** — compare monthly against paid media; organic should be 25-40% cheaper in mature programs
- **Organic share of total marketing pipeline** — target 20-35% for established B2B SaaS companies
- **Ranking coverage for high-intent keywords** — % of your ICP's top 50 buying-stage searches where you rank in positions 1-10
- **Traffic-to-lead conversion rate by cluster** — segment this to detect informational vs. buyer-intent traffic quality issues
- **Page decay detection accuracy** — how quickly does the system surface ranking declines before pipeline impact materializes

## Integration Tips

- **Google Search Console:** Export the "Queries" report (filtered to last 90 days) as CSV — this gives you impressions, clicks, CTR, and average position for every keyword. Combine with your CRM's "organic search" source filter to match keywords to pipeline. Use the "Pages" tab to get page-level click data.
- **HubSpot:** In HubSpot's "Traffic Analytics" report, filter by "Organic Search" source and segment by "Original Source Detail" to see which landing pages are generating organic MQLs. Export and paste directly into the Advanced Version template.
- **Salesforce + GA4:** Create a Salesforce report of all opportunities where Lead Source = "Organic Search," then use GA4's "Assisted Conversions" report to identify the landing pages that influenced those deals. The combination reveals true pipeline attribution.
- **Ahrefs / Semrush:** Use the "Content Gap" feature to instantly identify competitor keywords — export as CSV and filter to keywords with volume >500 and keyword difficulty <60 for quickest wins. Paste top 30 into the Keyword Gap section of the Advanced Version.
- **Zapier Automation:** Connect Google Search Console → Google Sheets (weekly auto-export) → Claude API (weekly analysis run using this prompt) → Slack (#seo-weekly channel). Gives your team an automated weekly SEO intelligence brief without manual reporting.
- **Looker Studio:** Build a live SEO revenue dashboard pulling from Google Search Console (via connector), HubSpot/Salesforce (via Supermetrics), and Google Analytics 4. Schedule weekly prompt-based analysis on top-performing and declining pages for the team.

## Troubleshooting

- **Problem:** Pipeline attribution data for organic search is unavailable — the CRM doesn't track lead source at the keyword level.
  **Solution:** Start with page-level attribution: create a HubSpot/Salesforce report filtering leads by "Original Source = Organic Search" AND "Landing Page = [your key SEO pages]." This approximates keyword-cluster pipeline contribution. For keyword-level attribution, implement UTM parameters on your Google Search Console → site links (use the "href" parameter trick) and run for 30 days before rerunning this prompt with real data.

- **Problem:** Organic traffic is growing but MQL volume from organic is flat or declining.
  **Solution:** This is the "traffic trap" problem the prompt is specifically designed to diagnose. The most common cause in B2B SaaS: successful ranking for informational keywords (what is X, how does X work) that attract researchers, not buyers. Run the Traffic Trap Detection section first. The fix is rarely "produce more content" — it's shifting production toward high-intent, bottom-of-funnel keywords (comparisons, alternatives, use case, pricing) where conversion rates are 5-10x higher.

- **Problem:** The AI recommends pausing content production for certain clusters, but the content team has ongoing editorial calendars and vendor relationships.
  **Solution:** Frame the recommendation as "redirect, not cut." Any editorial calendar slot freed from a low-ROAS cluster should be immediately reallocated to a Grade A cluster opportunity identified in the Keyword Gap analysis. Show the content team the pipeline math: 1 article targeting "revenue forecasting software comparison" at 3.2% visitor-to-lead rate × cluster ACV attribution = X expected pipeline per article. Make it concrete, and the editorial pivot becomes easy to justify.
