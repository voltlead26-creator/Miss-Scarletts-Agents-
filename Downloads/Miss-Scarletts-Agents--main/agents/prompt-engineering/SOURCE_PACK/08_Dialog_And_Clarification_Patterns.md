# Dialog and Clarification Patterns

Clarifying questions are useful only when the missing information changes the answer.

## Core facts

- Asking too early adds friction.
- Asking too late creates avoidable wrong answers.
- The useful question is the one that resolves real ambiguity.

## Practical guidance

- Ask only for missing data that materially changes the result.
- Prefer direct, concrete questions.
- Preserve the information already obtained.

## Common mistakes

- Asking every possible follow-up question.
- Skipping clarification when the answer depends on it.
- Losing earlier details during a long conversation.
