# Review Discipline

Prompt review should check whether the prompt is complete, testable, and aligned with the actual task.

## Review checklist

- Is the task unambiguous?
- Are the constraints explicit?
- Does the output format match the consumer?
- Is the relevant context preserved?
- Is the fallback behavior defined?
- Does the prompt have representative evaluation examples?

## Good signals

- The prompt is easy to reason about.
- The model is less likely to guess hidden rules.
- The prompt can be improved without rewriting the whole flow.
