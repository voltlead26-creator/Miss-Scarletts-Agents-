# Activation, Retention, and Conversion

These metrics answer different questions and should not be collapsed into one another.

## Core facts

- Activation measures the point where a user reaches meaningful initial value.
- Retention measures whether users return and continue to derive value.
- Conversion measures whether a target action was completed.
- Engagement measures how deeply or frequently the product is used.

## Practical guidance

- Define the activation milestone before building the onboarding dashboard.
- Use retention to understand repeat value.
- Use conversion when a sequence ends in a meaningful completion.

## Common mistakes

- Treating sign-up as activation by default.
- Using traffic as proof of retention.
- Comparing metrics that measure different stages of the journey.
