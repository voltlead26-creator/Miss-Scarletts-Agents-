# Color, Contrast, and Visual Perception

Visual design decisions affect whether users can see, distinguish, and understand content.

## Core facts

- Color alone should not convey meaning.
- Sufficient contrast is required for text and important UI boundaries.
- Status indicators should be backed by text, icons, or structure in addition to color.

## Practical guidance

- Test text contrast against its actual background.
- Do not rely only on hue differences for critical states.
- Make focus states visible and distinct.
- Consider how disabled, error, success, and warning states are distinguished.

## Common mistakes

- Using pale text on a low-contrast background.
- Indicating errors only with red.
- Making selected states too subtle to detect.
