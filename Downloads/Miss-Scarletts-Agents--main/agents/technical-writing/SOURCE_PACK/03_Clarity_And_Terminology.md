# Clarity and Terminology

Words matter because readers need the same term to mean the same thing every time.

## Core facts

- Terms should be introduced before they are reused.
- Acronyms should be expanded once and then used consistently.
- Terminology should not vary arbitrarily within the same product area.

## Practical guidance

- Use the same word for the same concept everywhere in the document set.
- Avoid synonym churn when precision matters.
- Prefer plain terms when they are accurate.

## Common mistakes

- Leaving abbreviations undefined.
- Using different labels for the same concept in different sections.
- Switching terminology for style instead of meaning.
