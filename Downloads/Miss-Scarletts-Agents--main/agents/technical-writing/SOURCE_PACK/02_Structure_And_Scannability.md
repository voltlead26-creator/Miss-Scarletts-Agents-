# Structure and Scannability

Readers scan before they read deeply, so structure must support that behavior.

## Core facts

- Short sections with clear headings improve findability.
- Lists and tables can make procedural or comparative content easier to absorb.
- Examples and recovery steps should be easy to locate.

## Practical guidance

- Put the most important information early.
- Break long explanations into smaller sections.
- Use heading levels consistently.

## Common mistakes

- Long paragraphs where a list would be clearer.
- Dense prose that hides the action the reader needs.
- Inconsistent heading structure.
