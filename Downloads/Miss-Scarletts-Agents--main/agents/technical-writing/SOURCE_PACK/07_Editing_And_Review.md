# Editing and Review

Editing should check more than grammar. It should check whether the reader can use the document.

## Core facts

- Editing for accessibility and clarity are related tasks.
- Consistency across a document set matters.
- A good review checks whether the document matches the target audience.

## Practical guidance

- Remove undefined terms and unnecessary words.
- Check that the document still works if read by a new user.
- Review error messages for cause, constraint, and fix.

## Common mistakes

- Editing only for style.
- Missing terminology drift.
- Ignoring whether the document actually helps the reader complete the task.
